Customize Me

Description

This is a generative web application. Use the interface on the right to make me look silly. Reload the page to start over.

Parameters

  • eyes: size of eyes
  • mouth: size of mouth
  • hairLength: length of hair
    hairColor: color of hair
    skinColor: color of skin
    eyeColor: color of eyes
    mouthColor: color of mouth
    hatColor: color of beanie
    bColor: color of background
